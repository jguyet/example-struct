# Team

## Hierarchy (execution chain)
1. project-manager
2. lead-developer
3. developer

## Agents
### project-manager
- Role: planification par itérations, découpage des blocks, DoD, ordre d’exécution
- Outputs: OBJECTIVE.md, TEAM.md, block/<id>.md

### lead-developer
- Role: décisions techniques, conventions, structure, choix stack/outils
- Outputs: décisions + guidelines + modifications nécessaires

### developer
- Role: implémentation concrète selon le block
- Outputs: code/files + notes + handoff
